# AWS-DCO-GenAI-Portfolio

## 1. 과정명
* **생성형 AI 활용 AWS DCO 직무 이해 및 학습 포트폴리오 제작 과정**

---

## 2. 포트폴리오 목적
본 포트폴리오는 생성형 AI와의 고도화된 프롬프트 협업을 통해 글로벌 데이터센터 운영(Data Center Operations, DCO) 직무의 핵심 언어, 인프라 개념, 그리고 표준 운영 절차(SOP)를 체계적으로 이해하기 위해 작성된 **학습 기록 저장소**입니다. 

직무 맥락을 바탕으로 가상의 장애 시나리오를 분석하고, CLI 및 스크립트를 활용한 교육용 업무 자동화 과정을 시뮬레이션하며 문제 해결력을 기르는 것을 목적으로 합니다.

---

## 3. 교육용 샘플 데이터 사용 안내
* **⚠️ 기밀 유지 가이드라인 준수:** 본 저장소에 포함된 모든 로그, 시나리오, 소스코드는 실제 특정 기업의 내부 운영 정보나 인프라 자산 데이터가 아닙니다.
* 공개된 직무 기술서(Job Description)와 대외 공개 기술 문서에 기반하여 가공된 **교육용 예시 데이터(`SAMPLE`, `DEMO`, `EDU`)**만을 사용하여 학습을 진행했습니다.

## 5. 보안상 포함하지 않는 정보
데이터센터 현장 보안 규칙의 중요성을 인지하고 있으며, 본 포트폴리오에는 아래와 같은 실제 정보나 민감한 데이터를 절대 포함하지 않습니다.
* **실제 장비 정보:** 실물 부품 일련번호(Serial Number), 비공개 장비 명칭, 하드웨어 스펙
* **실제 네트워크 정보:** 운영 환경의 실제 IP 주소, 내부 라우팅 경로, 실제 계정 및 인증 정보
* **실제 고객 및 운영 데이터:** 실제 고객사 명칭, 실제 서비스 티켓 번호, 내부 운영 회의록 등

## 6. 앞으로 추가할 산출물
* **주간 회고록 (Weekly Retrospective):** 각 실습 단계에서 발생했던 고충과 생성형 AI를 통해 교차 검증하며 해결한 과정을 담은 주간 학습 기록
* **나만의 직무 언어 확장 사전:** 인턴십 지원자로서 갖춰야 할 하드웨어 트러블슈팅 관련 신규 용어 및 다각적 원인 분석 가이드라인 추가
* **최종 포트폴리오 요약본:** 학습 성과를 시각적 흐름도로 정리한 최종 요약 리포트
